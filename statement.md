## Problem Statement

Managing lost and found items manually can lead to inefficiency, lost information, and frustration for users trying to recover their belongings. There is a need for an organized system to streamline reporting, tracking, and managing lost and found items, improving transparency and user satisfaction.

## Scope of the Project

This project provides a command-line interface application that enables users and administrators to report, view, update, search, and manage lost and found items. Data is persistently stored in JSON files, and the system supports essential functions such as claiming, returning, editing, and deleting items.

## Target Users

- Individuals who have lost items and want to report or search for them
- People who have found items and want to report or return them
- Administrators or staff managing the lost and found process in organizations such as schools, offices, libraries, or events

## High-Level Features

- Add records for lost and found items with details and contact info  
- List and search items by keywords in item names or locations  
- Mark items as claimed (lost) or returned (found)  
- Edit, delete, and undo deletion of item records  
- Persistent storage using JSON files  
- Interactive, menu-driven command-line interface for ease of use
