# vityarthi-project

Lost & Found Management System

Overview :- 
The Lost & Found Management System is a simple Python command-line application that helps manage items that are reported lost or found within a school, college, hostel, or public space.
It stores all data in a JSON file, allowing users to keep track of items with unique IDs and update their status.

Features :- 
1. Report Lost Item
Users can add an item along with the location where it was lost.
2. Report Found Item
Enter details of items that have been found.
3. View Lost/Found Items
Displays lists of all reported lost or found items.
4. Mark Lost Item as Claimed
Changes the status of a lost item once the owner retrieves it.
5. Persistent Storage
All data is saved automatically in a JSON file, so it remains available even after restarting the program.
